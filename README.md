# 📊 FNP Sales Analysis Dashboard (Excel)

## 📌 Project Overview
This project focuses on analyzing sales data from the FNP (Ferns N Petals) dataset using Microsoft Excel.  
An interactive dashboard was created to visualize business performance, customer behavior, and delivery efficiency.

---

## 🎯 Objectives
- Analyze overall sales performance  
- Identify top-performing products and occasions  
- Understand customer purchasing behavior  
- Evaluate delivery performance  
- Generate actionable business insights  

---

## 📊 Dashboard Features
- KPI Cards (Total Revenue, Total Orders, AOV, Delivery Delay %)  
- Revenue Analysis by Occasion and Category  
- Monthly Sales Trend (Line Chart)  
- Top Products and Top Customers  
- Top Cities by Revenue  
- Delivery Performance (On-Time vs Late)  
- Customer Segmentation (High vs Low Value)  
- Interactive Slicers (Date, Occasion)  

---

## 🛠️ Tools & Technologies Used
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Data Cleaning & Transformation  
- Basic Excel Formulas  

---

## 🧠 Key Insights
- Seasonal occasions significantly drive revenue  
- Top customers contribute a large share of total sales  
- Revenue is concentrated among a few high-performing products  
- Major cities generate the highest revenue  
- Most orders are delivered on time with minimal delays  
- Sales show strong seasonal trends during festive periods  

---

## 📂 Dataset
- Source: FNP Sales Dataset (Sample Data)  
- Contains order details, customer data, product info, and delivery data
